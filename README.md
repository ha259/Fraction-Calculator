📘 Fraction Arithmetic System in C++
📌 Overview

This project is a simple C++ application that implements a Fraction class to perform arithmetic operations on rational numbers.

It demonstrates key Object-Oriented Programming (OOP) concepts such as:

Constructors and destructors
Operator overloading
Encapsulation
Data normalization
⚙️ Features
Create fractions using different constructors:
Default (0/1)
Integer → fraction
Numerator & denominator
Automatic simplification using GCD (Euclidean algorithm)
Supports arithmetic operations:
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
Handles:
Negative values
Invalid denominator (error handling)
🧠 Concepts Used
C++ Classes & Objects
Operator Overloading (+, -, *, /)
Encapsulation (private attributes)
Euclidean Algorithm (GCD)
Input/Output streams
